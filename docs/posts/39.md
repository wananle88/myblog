### 循环语句
根据条件重复执行某段代码，通常需要结束条件，否则会死循环

### for

本质是重复执行一段代码，终止条件最常见的形式是一个计数值达到最终值，但通常用来处理次数已知的重复操作。

### while

也是重复执行，通常是未知边界，循环何时结束，取决于条件。


### 条件语句

### `if`与`else`
这些本质上属于布尔条件

`if`等于条件为真，也就是条件成立。

`else`等于条件为假，就是条件不成立。

伪代码
```
if x > 10 {
    // 条件成立 (true) → 走这里
} else {
    // 条件不成立 (false) → 走这里
}
```


### 总结
for循环，次数能提前定义

while循环，次数无法提前决定

if条件判断，为真时的处理

else条件判断，为假时的处理
